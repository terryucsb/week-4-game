Crystal Collector

Game Instructions


  1. A random number will appear at the start of the game that you have to match by clicking on different on crystals.
    Each crystals will have different values each trial of the game.
  2. There are four crystals below:
    • jake
    • kristin
    • charles
    • angela
  3. Clicking on each crystals will add different points to your total score.
  4. You win the game by matching your total score to the random number
  5. You have to click on the crystal to see the increase of value that each crystal holds
  6. Each time the game is restarted, the value of cyrstals will be different as well
